# GitHub Portfolio Starter

Это стартовая страница для GitHub Pages под ваше портфолио разработчика.

## Что уже подготовлено

- `index.html` — одностраничное портфолио
- `styles.css` — оформление
- `script.js` — плавная навигация по секциям

## Что заменить перед публикацией

1. При желании добавить ссылку на резюме PDF.
2. Добавить Telegram / email / LinkedIn.
3. Добавить YouTube-ссылку на robotics demo.
4. Добавить Figma-ссылки или превью.

## Как опубликовать через GitHub Pages

1. Создайте репозиторий `YOUR_USERNAME.github.io`
2. Загрузите в него файлы из этой папки
3. В `Settings -> Pages` убедитесь, что публикация идёт из ветки `main`
4. Сайт появится по адресу:

```text
https://YOUR_USERNAME.github.io/
```
